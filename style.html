4. Implement JavaScript Functionality:

In script.js, add functionality to fetch and display movie data:

javascript
document.getElementById('search').addEventListener('input', function() {
    const query = this.value;
    if (query.length > 2) {
        fetch(`https://api.themoviedb.org/3/search/movie?api_key=YOUR_API_KEY&query=${query}`)
            .then(response => response.json())
            .then(data => {
                const movies = data.results;
                const movieList = document.getElementById('movie-list');
                movieList.innerHTML = '';
                movies.forEach(movie => {
                    const movieElement = document.createElement('div');
                    movieElement.classList.add('movie');
                    movieElement.innerHTML = `
                        <img src="https://image.tmdb.org/t/p/w500${movie.poster_path}" alt="${movie.title}">
                        <h3>${movie.title}</h3>
                        <p>${movie.release_date}</p>
                    `;
                    movieList.appendChild(movieElement);
                });
            })
            .catch(error => console.error('Error fetching data:', error));
    }
});

5. Obtain an API Key:

To fetch movie data, you'll need an API key from The Movie Database (TMDb):

  • Visit TMDb's website and create an account.
  • Navigate to your account settings and find the API section to generate a new API key.
  • Replace YOUR_API_KEY in the fetch URL with your actual API key.

6. Test Your Website:

Open index.html in a web browser to test your website. As you type in the search bar, it should display movie results based on your query.
